Algorithm Overview

Huffman Coding

A bottom-up algorithm that builds an optimal prefix-free binary tree by repeatedly merging nodes with lowest frequencies.

O(n log n) time · O(n) space

Shannon–Fano Coding

A top-down approach that recursively divides symbols into groups of approximately equal probability.

O(n log n) time · O(n) space
